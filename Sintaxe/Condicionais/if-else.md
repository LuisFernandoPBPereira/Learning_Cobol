<h1 align="center">IF-ELSE</h1>

## Exemplo IF

```cobol
IF WRK-OPERACAO EQUAL "1"
    COMPUTE WRK-RESULTADO = WRK-NUMERO1 + WRK-NUMERO2
END-IF.
```

## Exemplo ELSE

```cobol
IF WRK-OPERACAO EQUAL "1"
    COMPUTE WRK-RESULTADO = WRK-NUMERO1 + WRK-NUMERO2
ELSE
    COMPUTE WRK-RESULTADO = WRK-NUMERO1 - WRK-NUMERO2
END-IF.
```
